# IPL-Analysis-Dashboard
Project 
🏏 IPL Data Analysis Dashboard | Power BI Project 📊

Excited to share my latest Power BI project where I analyzed historical Indian Premier League (IPL) data to uncover valuable insights using interactive dashboards and advanced DAX calculations.

🔍 Key Highlights:

✅ Interactive Season-wise Analysis
✅ Orange Cap & Purple Cap Holder Analysis
✅ Team Performance Comparison
✅ Points Table & Win Percentage
✅ Toss Decision vs Match Result Analysis
✅ Player Batting & Bowling Statistics
✅ Venue-wise Match Insights
✅ Head-to-Head Team Comparison
✅ Dynamic KPIs and Drill-through Reports
✅ Custom DAX Measures using CALCULATE, FILTER, SUMMARIZE, RANKX, LOOKUPVALUE, USERELATIONSHIP, and Variables

🛠️ Tools & Skills Used:
Power BI
Power Query
DAX
Data Modeling
Data Visualization
Interactive Dashboards
Sports Analytics

This project helped me strengthen my understanding of data modeling, DAX optimization, KPI design, and storytelling with data while transforming raw IPL datasets into meaningful business insights.
